Коннекторы
~~~~~~~~~~
Типичным средством поставки данных в платформу являются коннекторы.

Коннектор - специализированная программа, предназначенная для чтения данных с какого-либо источника и отправки их в платформу.

Предполагается, но не требуется, что один коннектор работает только с одним источником данных.

Общая логика работы коннекторов реализована в базовом модуле для создания коннекторов
`prs-connector-core <https://github.com/mp-co-ru/prs-connector-core>`__ и описана в документации к нему.

Для того, чтобы коннектор мог работать с платформой, в платформе должна быть создана запись,
содержащая настройки коннектора и список привязанных к нему тегов, для которых коннектор должен читать данные
из источника данных.

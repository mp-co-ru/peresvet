Новая функциональность именно в самой платформе:
1) Возможность работать с MQTT-источниками данных.
2) Работа с коннекторами переведана с протокола Websocket на MQTT.
3) "Интеграционные базы данных": возможность подключения сторонних баз данных.  Получение, запись, обновление данных в во внешних системах. Работа с такими внешними источниками через теги, как и с "собственной" исторической базой данных.
Таким образом обеспечивается интеграция разных информационных систем предприятия в рамках одной модели.
4) Получение данных тега через методы. Данные тега могут возвращаться не только из исторической базы данных системы, а быть результатом работы отдельного метода. Это обеспечивает возможность создавать модели технических объектов любой сложности, так как метод может быть реализован на любом языке программирования и возвращать данные клиенту как результат работы алгоритма любой сложности.
5) Новый "Конфигуратор": новое приложение для администратора.
6) Устранение ошибок первой версии.

Не платформа:
1) Новое ядро для создания коннекторов.
2) Коннектор для Modbus RTU/TCP.
3) Коннектор для OPC DA.
4) Коннектор для СПОДЭС
5) ИоТерра ОЕЕ в двух вариантах: для КЭАЗ и новая версия для Абрау-Дюрсо
6) ПАК ИоТерра для удалённого мониторинга электрических параметров
7) Инструмент для создания анимированных SVG-мнемосхем.
